Update app-nav component

The app-nav component is a shared component from the SKY UX default template used to display a navigation bar at the top of pages.

  1. Open the src/app/shared/app-nav directory.
  2. Create a app-nav.component.scss file, and add the following style: .sky-container { display: flex; }
  3. Open the app-nav.component.ts file, and add ./app-nav.component.scss to the styleUrls array. @Component({ selector: 'app-nav', templateUrl: './app-nav.component.html', styleUrls: ['./app-nav.component.scss'] })
