A blog layout lends well to sites with frequent blog posts or recurring meeting minutes. To achieve the blog format, use the list of components below to get started. For a live example, see the PAC Notes from the OCTO site. Sidebar Layout

Set as the default layout, the sidebar layout makes accessing different blog posts easy. When this layout is used, users can see the list of dates on the left side of the page.

Table of Contents

A table of contents is helpful for breaking up longer content into different sections, and it also provides user with a short overview of the post. When a user selects an item from the table of contents, they're taken to that page anchor.

To generate a table of contents, the page must also use the Page Anchor component. Page Anchor

Page anchors are used in conjunction with the Table of Contents and provide users with an easy way to move around your page.

Text Expand

If you'd like to control the amount of content displayed under each anchor, use the Text Expand component to hide content after a specific number of characters.
